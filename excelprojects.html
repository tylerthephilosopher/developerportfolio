Sales Dashboard

Sample sales dashboard from a sample dataset I found online. I will show my process from start to end. First step is to get the data we are trying to analyse. Once data is collected we open it up in excel. Once in excel , it is best advised to open in power query to check all columns and make sure the proper values for each column, as well as find any missing data and how to fix it. This example unfortunately has all the data so we do not need to change anything.

Now that we have some data to work with the next part is to start a new page to use as our dashboard.

Sales by Productline

Our first pivottable is total sales by productline. An easy visual for this information is to use a bar graph.

Sales by Top 5 Countries

The second pivot table is top 5 performing countries, using the value filter. This allows us to see the top five from all of the countries.

Map of all sales

The third pivot table is a map of all of the sales, to be able to get data to display a map, I neeeded to put the data outside the pivot table. Once outside you need enough information to be able to accurately depict the information. We need the two fields, country and state or city, to narrow down the options.

Final Product

And finally after all of our pivot tables to graphs, we can copy and paste the graphs to our dashboard
